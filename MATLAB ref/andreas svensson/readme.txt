***************************************************************

                    Matlab code for for
          A flexible state space model for learning
                nonlinear dynamical systems
		   
***************************************************************
VERSION:
  
  0.1 First public version, made available May 18, 2016.
  

INTRODUCTION:

  This package contains the code used in the paper
  'A flexible state space model for learning nonlinear dynamical systems' 
  available as pre-print on arXiv.org: http://arxiv.org/abs/1603.05486

  
PACKAGE CONTENTS:

* All examples from the paper, each in a separate subfolder

* An introduction to the code, meant to walk through line by line for the interested user


NOTES:

* The code is developed using Matlab R2015a (8.5)
